注意

「スクリプト機能」はNCMBの無料プランの場合、月間100回までしか使えません。
スクリプトを本格的に使う場合は、優勝会員になる必要があります。



使い方

１．gachaScript.jsの次の箇所にAPIキーをそれぞれ設定する  

  var app_key = 'YOUR_APP_KEY';
  var client_key = 'YOUR_CLIENT_KEY';

2.　データストアにクラス「GachaSettings」を作り、
フィールド「ID」「NN」「NR」「SR」「UR」を作る。
「ID」には"1"、その他にはカードのレアリティに合わせて確立調整する。

3.gachaScript.jsをスクリプト機能にアップする。
